# Aetheris – Phase 8: Intelligent Context Builder & Memory Relevance Filtering

You are a Principal AI Systems Architect.

This is a core architectural improvement.

Do NOT redesign Aetheris.

Do NOT modify the Memory Service, Reflection Engine, Cognitive Request Router, or Provider Manager.

Your task is to redesign ONLY the Context Builder so that it injects **only relevant memories** into the LLM prompt.

---

# Problem

The current Context Builder blindly sends retrieved memories to the LLM.

This causes unrelated memories to appear in responses.

Example

User

"What is 5 + 5?"

Retrieved Memories

* Building Aetheris
* Favorite Language = Python
* Likes Pizza
* Swimming

Current Response

The LLM unnecessarily talks about Aetheris, Python, pizza, and swimming.

This is incorrect.

---

# Objective

The Context Builder must become intelligent.

It must determine whether each retrieved memory is relevant to the current user request.

Only relevant memories may be included in the final prompt.

---

# Memory Relevance Filter

Create:

backend/app/services/memory_relevance_filter.py

Responsibilities:

* Receive user message
* Receive retrieved memories
* Score each memory for relevance
* Remove irrelevant memories
* Return only useful memories

---

# Relevance Scoring

Each memory receives a relevance score.

Factors:

* Semantic similarity
* Intent match
* Memory category
* Recency
* Memory strength
* User request type

Final score:

0.0 → 1.0

---

# Threshold

Default relevance threshold

0.65

If relevance < threshold

Discard the memory.

Do not include it in the prompt.

# Context Rules

## Math

Example

"What is 5 + 5?"

Inject:

No memories.

---

## Greeting

"Hi"

Inject:

No memories.

---

## Programming

"Explain recursion."

Inject:

Programming preferences only if relevant.

Ignore food, hobbies, names, etc.

---

## Personal Memory

"What is my favorite language?"

Inject:

Favorite language memory only.

---

## Project

"What project am I building?"

Inject:

Project memories only.

---

## Food

"What food do I like?"

Inject:

Food memories only.

---

# Context Size
